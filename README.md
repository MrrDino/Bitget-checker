# УСТАНОВКА ПРОЕКТА

1. Скачайте репозитрой с GitHub:

    ```https://github.com/MrrDino/Bitget-checker.git```
   
3. Перейдите в репозиторий с кодом:

    ```cd Bitget-checker```
   
3. Откройте файл variables.env и внесите туда id чата и токен бота:

    ```TOKEN= CHAT_ID=```

4. Поднимите докер:

    ```docker-compose up -d```
