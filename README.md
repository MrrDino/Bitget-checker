# УСТАНОВКА ПРОЕКТА

1. Скачайте репозитрой с GitHub:

```git clone https://github.com/MrrDino/Stargate-swap.git```

2. Создайте файл variables.env и внесите туда id чата и токен бота:

```touch variables.env```

```Названия переменных: TOKEN, CHAT_ID```

3. Поднимите докер:

```docker-compose up -d```
